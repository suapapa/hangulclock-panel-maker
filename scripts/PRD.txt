# Overview  
이 프로젝트는 한글시계의 패널 이미지를 웹에서 생성할 수 있는 웹앱입니다. 기존의 _ref/make_panel.py 스크립트의 기능을 웹으로 옮겨, 사용자가 다양한 설정을 조정하며 실시간으로 패널 이미지를 만들 수 있도록 합니다. 한글시계 패널을 제작하려는 메이커, 디자이너, 개발자를 주요 대상으로 하며, 로컬 환경에 의존하지 않고 브라우저에서 바로 결과물을 얻을 수 있다는 점이 큰 장점입니다.

# Core Features  
- 패널 이미지 생성: 한글시계 패널 이미지를 웹에서 바로 생성
- 자동 글꼴 크기 계산: 패널 크기에 맞춰 글꼴 크기를 자동으로 산출
- 다양한 설정값 조정: 패널 크기, 글자판 마진, 각 글자 마진, y 오프셋 등 주요 파라미터를 실시간으로 조정 가능
- 기본값 제공: 모든 설정값에 합리적인 기본값 제공
- 이미지 다운로드: 생성된 이미지를 PNG 등으로 다운로드 가능
- 실시간 미리보기: 설정 변경 시 즉시 결과 반영

# User Experience  
- 주요 사용자: 한글시계 DIY 메이커, 패널 제작자, 개발자
- 주요 플로우: 웹 접속 → 설정값 입력/조정 → 실시간 미리보기 확인 → 이미지 다운로드
- UI/UX: 직관적인 슬라이더/입력창, 실시간 미리보기, 모바일/데스크탑 대응

# Technical Architecture  
- 프론트엔드: Svelte. Canvas API 활용
- 백엔드: 없음(모든 기능을 클라이언트에서 처리)
- 주요 컴포넌트: 설정 입력 폼, 미리보기 패널, 이미지 생성/다운로드 모듈
- 데이터 모델: 패널 설정값(크기, 마진, 오프셋 등), 글자판 문자열, 폰트 파일

# Development Roadmap  
- 1단계(MVP):
  - 기본 UI(설정 입력, 미리보기, 다운로드)
  - 패널 이미지 생성 로직 이식 및 Canvas 렌더링
  - 자동 글꼴 크기 계산
  - 주요 설정값 조정 기능
- 2단계(고도화):
  - 다양한 글꼴 지원 및 업로드
  - 설정값 프리셋 저장/불러오기
  - 모바일 UI 최적화
  - 추가 이미지 포맷 지원

# Logical Dependency Chain
- 패널 렌더링 로직 → 설정값 입력 UI → 실시간 미리보기 → 이미지 다운로드
- MVP: 최소한의 설정값 조정 및 렌더링 → 고도화: 폰트 업로드, 프리셋 등 부가 기능

# Risks and Mitigations  
- Canvas에서 한글 폰트 렌더링 호환성 문제 → 주요 브라우저 테스트, 기본 폰트 내장
- 대용량 폰트 파일 처리 → 파일 크기 제한, 기본 폰트 우선 제공
- 모바일 환경에서 UI 깨짐 → 반응형 디자인 적용
